## Phase 3: Introduce Abstractions (ABCs / Protocols)

### 3.1: `InteractiveButton` ABC
All 4 canvas buttons (RoundedButton, IconButton, MiniToolButton, WebSearchToggleButton) share identical hover/press/active/disabled state machine. Extract common base:
```python
class InteractiveButton(tk.Canvas, ABC):
    @abstractmethod
    def _draw(self) -> None: ...
    @abstractmethod
    def _hit_test(self, x: int, y: int) -> bool: ...
    # Common: _on_enter, _on_leave, _on_press, _on_release, _set_state, redraw
```

### 3.2: `HtmlExtractor` Protocol
All 5 HTML parsers follow the same pattern (feed HTML → extract data). Protocol:
```python
class HtmlExtractor(Protocol):
    def feed(self, html_data: str) -> None: ...
    def get_result(self) -> dict[str, Any]: ...
```

### 3.3: `MaterialProcessor` ABC
All material types (web, youtube, codepen, github, video) follow intake → process → write vault pattern:
```python
class MaterialProcessor(ABC):
    @abstractmethod
    def can_handle(self, kind: str) -> bool: ...
    @abstractmethod
    def process(self, item: dict) -> MaterialRoutingReport: ...
```

### 3.4: `MarkdownRenderer` Protocol
15+ `render_*_markdown()` functions share structure:
```python
class MarkdownRenderer(Protocol):
    def render(self, data: dict) -> str: ...
```

### 3.5: `BookCatalogClient` ABC
Open Library + Google Books lookup:
```python
class BookCatalogClient(ABC):
    @abstractmethod
    async def search_by_isbn(self, isbn: str) -> list[dict]: ...
    @abstractmethod
    async def search_by_title_author(self, title: str, author: str) -> list[dict]: ...
```

## Phase 5: Refactor KnowledgeChatApp (the God class)

### Current: ~4,750 lines, handles everything
### Target: ~800-1000 lines, delegates to extracted modules

The refactored `KnowledgeChatApp` keeps only:
- Tkinter window setup and layout (`__init__`, `build_ui`)
- Widget creation (delegates to `ui/`)
- Event handlers (thin wrappers that call into extracted modules)
- Session management (thin delegation)
- Settings dialog (UI only, delegates save/load to config)

All domain logic moves to the extracted modules. The class becomes a **coordinator**, not a monolith.
